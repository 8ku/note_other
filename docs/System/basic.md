## 计算机数据大小

### 基本单位

1B(Byte) = 8 bit

1KB(Kilobyte) = 1024 B

1MB(Megabyte) = 1024 KB

1GB(Gigabyte) = 1024 MB

1TB(Terabyte) = 1024 GB

1PB(Petabyte 拍字节) = 1024 TB

1EB(Exabyte 艾字节) = 1024 PB

1ZB(Zettabyte 泽字节) = 1024 EB

1YB(Yottabyte 尧字节) = 1024 ZB

1BB(Brontobyte) = 1024 YB

### 基本换算

1 MB = 1M Byte = 8M bits

10 MB = 80Mb
