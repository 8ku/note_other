# Blender Sculpting



## 快捷键

- Remesh
  - shift+r：设置 Voxel Size
  - command+r/ctrl+r：应用 remesh

- 雕刻模式下在多个物体中切换：alt/option+q 左键点击物体
- <span class="highlight">把合并的物体组合成一个物体：Remesh</span></span>
